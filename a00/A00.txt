🧠 ANOTAÇÕES LIVRES:
------------------------------------------------------------
Comando para criar aplicação com Expo:

npx create-expo-app@latest --template

Explicações:
- `@latest`: garante que será usada a versão mais recente do pacote
- Após rodar o comando, escolha o template "Blank"
- Nome do app pode ser definido na hora (ex: `my-first-app-test`)
- Após a criação, o terminal sugere rodar `npm install` para instalar dependências

------------------------------------------------------------

Comando para iniciar o projeto Expo após criado:

npx expo start

- Esse comando abre o Metro bundler e permite escolher emulador, browser ou dispositivo físico.
------------------------------------------------------------
Variavel de ambiente = uma vez configurada, toda vez que o computador é reiniciado, a variavel é recriada com um valor
